#  Banking Information System (Core Java)

## Project Overview
This project is a console-based Banking System developed using Core Java. It simulates real-world banking operations with basic user interaction.

---

## Features
- User Registration
- Account Creation with Unique ID
- Deposit & Withdrawal
- Fund Transfer
- Account Statement (Transaction History)
- Basic Login Authentication
- Error Handling

---

## Technologies Used
- Java (Core)
- OOP Concepts
- ArrayList / File Handling (if used)
